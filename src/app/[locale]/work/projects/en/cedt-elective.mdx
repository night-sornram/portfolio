## Overview

Develop an elective course selection system for CEDT students.

## Key Features

- **Course Ranking**: Ranking of elective course
- **Course List**: View all elective courses
- **Course Interested**: bookmark elective course


## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **Prisma**: For backend 
- **Postgresql**: For database 

## Outcome

This system can be used to select elective subjects for CEDT students in the second semester of the 2024 academic year.
