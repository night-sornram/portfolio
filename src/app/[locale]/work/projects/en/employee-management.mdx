## Overview

Develop a system for checking in/out and requesting leave for employees.

## Key Features

- **Check In**: Employees can check in of work.
- **Check Out**: Employees can check out of work.
- **Leave Management**: Employees can request leave from work.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **Gofiber**: For backend 
- **Postgresql**: For database
